# med-tools

自包含、純前端的臨床輔助小工具集。皆為單一 HTML 檔，無後端、無追蹤，手機/iPad 可離線開啟。

線上：https://shurueiwu.github.io/med-tools/

## 工具

| 工具 | 說明 |
|---|---|
| [BSA 體表面積估算](BSA_calculator.html) | 滑鼠/手指塗色估計 %TBSA，採 Lund-Browder 分區年齡加權（嬰兒～成人 6 欄），可塗可擦。 |

## 維護

各工具的 master（SSOT）保存在本機 `~/Documents/私人資料/`；本 repo 為發布副本。更新流程：改 master → 複製到本 repo → commit + push。
